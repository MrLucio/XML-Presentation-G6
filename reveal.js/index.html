XML Schema

Sviluppato dal World Wide Web Consortium

Funziona come un DTD

Ma si basa su XML
È quindi analizzabile da un parser XML

Con XML Schema possiamo definire:

  • elementi
  • attributi
  • ordine dei tag
  • tipi di dati
  • e molto altro

Estensibile

e

riutilizzabile

Si possono definire diversi tipi di dato

  • integer
  • float
  • string
  • date
  • ...

oppure

È possibile applicare delle restrizioni sui dati

Esempio di un file XML


						
						
							Intel 8080
							64mb
							M320x
							YeetPower 20W

Esempio del suo file XSD


							<?xml version="1.0" encoding="utf-8" ?>
							<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema">
							    <xs:element name="computer" type=computerType />
							    <xs:complexType name="computerType">
							    <xs:sequence>
							        <xs:element name="cpu" type=xs:string>
							        <xs:element name="ram" type=xs:string>
							        <xs:element name="mobo" type=xs:string>
							        <xs:element name="psu" type=xs:string>
							    </xs:sequence>
							    </xs:complexType>
							</xs:schema>
						

							<?xml version="1.0" encoding="utf-8" ?>
							<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema">

							        <xs:element name="psu" type=xs:string>
							    </xs:sequence>

							</xs:schema>

Collegare un file XML a uno schema XSD


						<?xml version="1.0"?>
						<computer
						  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
						  xsi:noNamespaceSchemaLocation="http://mysite.it/msg.xsd">
						  <cpu>Intel 8080</cpu>
						  <ram>64mb</ram>
						  <mobo>M320x</mobo>
						  <psu>YeetPower 20W</psu>
						</computer>

L'attributo va inserito all'interno del tag radice

Questo definirà il namespace

ma non validerà automaticamente il documento XML!

Il documento XML associato allo schema prende il nome di documento istanza

Tipi di elementi e attributi

semplici

  • primitivi
  • derivati

complessi

  • dsa
  • eqw
Gli elementi possono essere di tipo semplice o complesso mentre gli attributi possono essere solo di tipo semplice
