XML Schema

Sviluppato dal World Wide Web Consortium

Funziona come un DTD

Ma si basa su XML
È quindi analizzabile da un parser XML

Con XML Schema possiamo definire:

  • elementi
  • attributi
  • ordine dei tag
  • tipi di dati
  • e molto altro

Estensibile

e

riutilizzabile

Si possono definire diversi tipi di dato

  • integer
  • float
  • string
  • date
  • ...

oppure

È possibile applicare delle restrizioni sui dati

Collegare un file XML a uno schema XSD


						<?xml version="1.0"?>
						<computer
						  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
						  xsi:noNamespaceSchemaLocation="http://mysite.it/msg.xsd">
						  <cpu>Intel 8080</cpu>
						  <ram>64mb</ram>
						  <mobo>M320x</mobo>
						  <psu>YeetPower 20W</psu>
						</computer>

L'attributo va inserito all'interno del tag radice

Questo definirà il namespace

ma non validerà automaticamente il documento XML!

Il documento XML associato allo schema prende il nome di documento istanza

Gli elementi possono essere di tipo semplice o complesso mentre gli attributi possono essere solo di tipo semplice

Un tipo di dato consiste di:

  • uno spazio di valori
  • uno spazio lessicale
  • un insieme di facet

Spazio di valori

L'insieme dei valori che un certo dato può assumere

Spazio lessicale

L'insieme delle stringhe che rappresentano un valore

Insieme di facet

Utilizzate per restringere valori del tipo base

Tipi predefiniti

  • string
  • decimal
  • float
  • double
  • boolean
  • dateTime
  • date
  • time

Derivare un tipo semplice

e restringere il suo campo di valori


						<xs:simpleType name="punteggio">
						   <xs:restriction base="integer">
						      ...facets
						   </restriction>
						</xs:simpleType>

I facets restringono un tipo di dato

e ogni tipo di dato può essere ristretto in modo diverso

Tipologie di restrizioni:

  • Intervalli, numerici e di date
  • Limitazioni di lunghezza sulle stringhe
  • Rappresentazioni specifiche di tipi numerici
  • Enumerazioni
  • Vincoli di restrizioni tramite pattern (regex)

Facets - Intervalli numerici

Exclusive


							<xs:simpleType name="punteggio">
							    <xs:restriction base="integer">
								<xs:minExclusive value="0" \>
								<xs:maxExclusive value="100" \>
							    </restriction>
							</xs:simpleType>

Inclusive


							<xs:simpleType name="punteggio">
							    <xs:restriction base="integer">
								<xs:minInclusive value="0" \>
								<xs:maxInclusive value="100" \>
							    </restriction>
							</xs:simpleType>
